Hello GitHub Wold

Initial Step:

  • git init
  • git add .
  • git git commit -m "comment"
  • git remote add origin path_link
  • git push -u origin master

Modify Step:

  • git add .
  • git git commit -m "modify comment"
  • git push
  • git diff (to show difference code)

Collaborate Step:

  • git pull

Branch Step:

  • git branch
  • git checkout -b branch_name
  • git checkout -d branch_name
  • git checkout master
